# Comandos vigentes

Actualizado: 6 de marzo de 2026

## Desarrollo

```powershell
npm.cmd run dev
```

URL local: `http://localhost:3000`

## Build y validacion

```powershell
npm.cmd run css:guard
npm.cmd run build
```

## Ejecutar modo produccion local

```powershell
npm.cmd run preview
```

## Levantar build ya generado

```powershell
npm.cmd run start
```

## Smoke test rapido

```powershell
npm.cmd run test:smoke
```

## Si PowerShell bloquea npm

```powershell
cmd /c npm.cmd run dev
```
